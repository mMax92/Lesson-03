Example for repository

  1. git config --global user.name "Your Name"
  2. git config --global user.email "your_email@whatever.com"
  3. git init (создать репозиторий (один раз для проекта))
  4. git add index.html
  5. git commit -m "Создал index.html"
  6. git log (просмотр истории коммитов)
  7. git checkout (переключиться на коммит)
  8. git status (посмотеть статус репозитория)
  9. git remote add origin https://...
  10. git push origin master
